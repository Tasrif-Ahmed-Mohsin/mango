# AgriCommerce

A scalable e-commerce website for agriculture products directly from farmers to customers.

## Features
- Dynamic category-based system (Mangoes, Bananas, etc.)
- Shopping Cart functionality with local storage persistence
- Farm-to-table UI design using TailwindCSS
- Fully responsive on mobile devices

## Getting Started

1. `npm install`
2. `npm run dev` (to run the app in development)
3. Open `http://localhost:3000`

## Tech Stack
- Next.js (App Router)
- React
- TailwindCSS
- TypeScript
